# Programming Quiz App

A lightweight, browser-based quiz application that tests programming knowledge with randomized multiple-choice questions. Built with **HTML**, **CSS**, and **JavaScript**, it uses the [Open Trivia Database](https://opentdb.com/) API to deliver quiz content and [SweetAlert](https://sweetalert.js.org/) for user-friendly alerts.

## Overview

Enter the number of questions you want to attempt, start the quiz, and answer each question one by one. The app tracks your score live, highlights correct and incorrect answers, and includes a clean responsive interface for desktop and mobile screens.

## Features

- Select between 1 and 50 questions
- Randomized multiple-choice programming questions
- Live score tracking
- Clear answer feedback with visual highlighting
- Responsive layout for different screen sizes
- Restart option for replaying the quiz
- Built without a frontend framework

## Tech Stack

- HTML
- CSS
- JavaScript
- Open Trivia Database API
- SweetAlert2

## Getting Started

### Clone the repository

```bash
git clone https://github.com/RISTONRODZ/Programming-Quiz-App.git
cd Programming-Quiz-App
```

### Run the app

Open `index.html` in your browser, or serve the project with a local development server if you prefer.
